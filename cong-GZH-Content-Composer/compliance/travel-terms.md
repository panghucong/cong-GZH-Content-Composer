# 旅游行业L1合规词库

## 一、广告法敏感词（旅游特定）

| 风险等级 | 敏感词/表述 | 建议替换 |
|---------|-----------|---------|
| 🔴 高 | 最佳旅游地、最美景点、第一度假村 | "热门旅游地"、"推荐景点" |
| 🔴 高 | 唯一、绝无仅有 | "独具特色"、"别具一格" |
| 🔴 高 | 零差评、百分百满意 | "好评率XX%"、"多数游客反馈良好" |
| 🟡 中 | 顶级酒店、奢华体验 | "高端酒店"、"品质住宿" |
| 🟡 中 | 必去景点、人生必去 | "推荐景点"、"热门打卡地" |
| 🟢 低 | 热门目的地、网红打卡 | 无需替换 |

## 二、价格/费用误导

| 风险等级 | 敏感词/表述 | 建议替换 |
|---------|-----------|---------|
| 🔴 高 | 全程免费（隐藏自费项目） | "基础费用包含XX（自费项目详见行程单）" |
| 🔴 高 | 最低价、全网最低 | "优惠价"、"限时特惠" |
| 🔴 高 | 一价全包（未含所有费用） | "费用包含：XX（不包含XX）" |
| 🟡 中 | 超值套餐、白菜价 | "性价比套餐" |
| 🟡 中 | 买一送一（有条件限制） | "两人同行一人免单（限XX日期）" |

## 三、行程/服务夸大

| 风险等级 | 敏感词/表述 | 建议替换 |
|---------|-----------|---------|
| 🔴 高 | 保证看到XX景观（自然景观） | "大概率可观赏到XX景观（受天气等自然条件影响）" |
| 🔴 高 | 全程五星级酒店（混住四星） | "全程入住XX酒店或同级别酒店" |
| 🟡 中 | 纯玩无购物（实际有） | "无强制购物"、"自由购物" |
| 🟡 中 | 深度游（走马观花） | "XX天行程，涵盖XX个景点（具体以行程单为准）" |
| 🟢 低 | 网红打卡、出片率高 | 无需替换 |

## 四、安全相关

| 风险等级 | 敏感词/表述 | 建议替换 |
|---------|-----------|---------|
| 🔴 高 | 绝对安全、零风险 | "配备专业导游/领队"、"购买旅游意外险" |
| 🟡 中 | 适合所有人、老少皆宜 | "适合XX年龄段的游客" |
| 🟡 中 | 包赔、全赔 | "建议购买旅游保险（具体以保险条款为准）" |
