## 一、平台算法适配（新增）

### 1.1 微信公众号算法机制

**核心机制**：
```
推荐逻辑：订阅+朋友圈分享+看一看推荐
          ↓
关键指标：打开率→分享率→完读率→在看率
          ↓
算法偏好：高打开率（>15%）+ 高分享率（>8%）+ 长尾传播
```

**优化策略**：
1. **标题优化**：悬念+数字+情绪（提升打开率）
   - ✅ "3个理由，劝你一定要买重疾险"
   - ❌ "重疾险产品介绍"

2. **开篇优化**：前3行必须抓住注意力（提升完读率）
   - 提问式："你有没有想过..."
   - 故事式："上个月，我朋友..."

3. **分享优化**：文末添加"关注引导"+"在看引导"
   - "如果觉得有用，点个『在看』告诉更多人"

4. **长尾优化**：关键词布局（提升『看一看』推荐）
   - 文中自然植入3-5个行业关键词

## 三、平台热点追踪（新增）

### 3.1 热点发现机制

**工具组合**：
1. **百度指数**：查看关键词搜索趋势
2. **微信指数**：查看微信生态热度
3. **微博热搜**：查看实时热点话题
4. **抖音热点**：查看短视频热点
5. **知乎热榜**：查看深度讨论热点

**操作流程**：
```
Step 1：每天上午10点，查看5大平台热搜
  ↓
Step 2：筛选与行业相关的热点（如"医保改革"）
  ↓
Step 3：判断热点类型（长期热点/短期热点/突发热点）
  ↓
Step 4：制定蹭热点策略（深度解读/快速回应/情感共鸣）
```

---

### 3.2 热点蹭用策略

#### 类型A：长期热点（如"老龄化"）
**策略**：深度解读+专业分析
**示例**：
```
热点：老龄化加速
→ 保险角度："老龄化时代，我们该如何规划养老？"
→ 医疗角度："老龄化时代的健康管理指南"
→ 金融角度："老龄化下的理财新思路"
```

#### 类型B：短期热点（如"某明星理赔"）
**策略**：快速回应+情感共鸣
**示例**：
```
热点：某明星患癌理赔
→ 保险角度："从XX理赔案，看保险的重要性"
→ 情感角度："生命如此脆弱，我们能做什么？"
```

#### 类型C：突发热点（如"某疾病爆发"）
**策略**：专业科普+权威引用
**示例**：
```
热点：某传染病爆发
→ 医疗角度："XX病，你需要知道的5件事"
→ 保险角度："XX病，保险赔不赔？"
```

---

### 3.3 热点风险规避

**避坑清单**：
1. ❌ 不蹭政治敏感热点
2. ❌ 不蹭悲剧性热点（如空难、地震）
3. ❌ 不蹭争议性热点（如明星八卦）
4. ❌ 不恶意营销热点（如灾难营销）
5. ✅ 只蹭正相关性热点（与行业相关+正向价值）

**合规检查**：
- 热点与品牌形象是否匹配？
- 蹭热点是否违反广告法？
- 蹭热点是否会引起用户反感？

### 7.2 标签生成规则

1. **从标题中提取核心关键词** → 生成核心标签

3. **调用WebSearch获取热点话题** → 生成热点标签
4. **分析用户痛点** → 生成情感标签
5. **分析使用场景** → 生成场景标签
6. **组合成#标签#格式** → 按平台规则放置
