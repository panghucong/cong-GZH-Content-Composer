# 模板：金融理财（Q5）

> **模板编号**：T-FIN-001
> **适用场景**：金融理财知识科普、投资指南
> **预设参数**：I3/C1/F4(P2/T6/U4)

---

## 模板结构

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | "{数字}%的{人群}都在犯的{理财错误}" | "80%的职场人都在犯的理财错误" |
| 数字型 | "用{数字}个方法，让你的{资产}增长{X}%" | "用3个方法，让你的存款多赚20%" |
| 情绪型 | "别让你的钱{贬损方式}了" | "别让你的钱在银行里贬值了" |

### 正文结构（F4：正反辩证框架）

```
【开头钩子】（数据型/痛点型）
"最新数据显示..." / "你是不是也觉得..."

【现状分析】（问题揭示）
- 数据：{行业数据}
- 问题：{常见理财问题}
- 影响：{对用户的影响}

【方案对比】（多方案分析）
- 方案A：{方案名称}
  - 优点：{优点}
  - 缺点：{缺点}
  - 适合人群：{人群}
- 方案B：{方案名称}
  - 优点：{优点}
  - 缺点：{缺点}
  - 适合人群：{人群}
- 对比总结：{一句话对比}

【行动指南】（可执行步骤）
- 步骤1：{具体步骤}
- 步骤2：{具体步骤}
- 步骤3：{具体步骤}
- 注意事项：{风险提示}

【结尾引导】
关注我，每周分享理财干货。你现在的理财方式是什么？留言交流。
```

### 金句库

| 类型 | 金句 |
|------|------|
| 痛点型 | "最危险的理财方式，就是把钱放在那里什么都不做" |
| 数据型 | "据统计，通胀每年吞噬约{X}%的购买力" |
| 情感型 | "理财不是有钱人的专利，而是每个人的必修课" |
| 行动型 | "今天开始理财，未来的你会感谢现在的自己" |

### 合规检查点

| 检查项 | 规则 |
|--------|------|
| 收益承诺 | ❌ 禁止"保证收益"/"稳赚不赔"/"年化XX%" |
| 投资建议 | ❌ 禁止直接推荐具体股票/基金代码 |
| 风险提示 | ✅ 必须包含"投资有风险，入市需谨慎" |
| 数据来源 | ✅ 所有数据必须标注来源 |
| 免责声明 | ✅ 必须包含"本文仅供参考，不构成投资建议" |
