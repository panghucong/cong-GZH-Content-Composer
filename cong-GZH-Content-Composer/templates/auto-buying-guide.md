# 汽车选购指南模板（T-AUTO-001）

> **模板类型**：测评对比类
> **适用场景**：车型对比/购车攻略/用车成本分析
> **质量提升**：+10-15%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[车型A] vs [车型B]：到底哪个更值得买？
```

### 1.2 变体公式
```
① "[车型名称]值得买吗？3个理由告诉你"
② "购车攻略｜[预算]元，[品牌]还是[品牌]？"
③ "[车型A]、[车型B]、[车型C]全面测评"
④ "买车避坑｜这3个坑，90%的人都踩过"
```

### 1.3 关键词库
- 购车词：买车、购车、车型、配置、试驾
- 对比词：vs、对比、测评、区别、哪个好

## 二、正文结构（4段式）

### 2.1 开篇：提出问题（10-15%）
**目标**：让读者意识到"我需要选择，但不知道选哪个"

**模板**：
```
最近有很多朋友问我：
"[车型A]和[车型B]，到底买哪个？"
"网上测评五花八门，越看越迷糊。"
"怕买错，又怕买亏。"

今天，我就用一张表，
把[车型A]和[车型B]的区别讲清楚。
看完这篇，你就知道该怎么选了。
```

---

### 2.2 主体：车型对比（60-70%）
**目标**：用表格/列表清晰展示区别

**对比维度**：
```
## [车型A] vs [车型B]：核心区别

| 对比维度 | [车型A] | [车型B] | 哪个好？ |
|---------|---------|---------|---------|
| **价格** | [描述] | [描述] | [结论] |
| **动力** | [描述] | [描述] | [结论] |
| **配置** | [描述] | [描述] | [结论] |
| **空间** | [描述] | [描述] | [结论] |
| **油耗** | [描述] | [描述] | [结论] |
| **保养成本** | [描述] | [描述] | [结论] |
| **保值率** | [描述] | [描述] | [结论] |

**我的建议**：
- 如果你[条件A]，选[车型A]
- 如果你[条件B]，选[车型B]
- 如果你[条件C]，两个都不推荐，建议考虑[车型C]
```

---

### 2.3 转折：避坑指南（15-20%）
**目标**：告诉用户"买车要注意什么"

**模板**：
```
## 买[车型类型]，这3个坑别踩

❌ **坑1：[常见误区1，如只看外观不看动力]**
→ 正确做法：[建议]

❌ **坑2：[常见误区2，如忽略保养成本]**
→ 正确做法：[建议]

❌ **坑3：[常见误区3，如轻信销售优惠]**
→ 正确做法：[建议]

**记住一句话**：[总结性建议]
```

---

### 2.4 结尾：行动召唤（5-10%）
**目标**：引导用户咨询/试驾

**模板**：

**承诺**：不推销，只给真话。

## 三、金句库

- "适合别人的，不一定适合你。"
- "对比不是为了分高下，而是为了帮你看清自己。"

### 3.3 结尾金句
- "理性购车，从容出行。"
- "今天的选择，决定未来5年的驾驶体验。"

---

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "最好"、"第一"、"唯一"（绝对化用词）
- ❌ "零油耗"、"永不故障"（夸大性表述）
- ❌ 贬低竞品车型（违反广告法）

### 4.2 推荐表述
- ✅ "根据您的需求，[车型A]更适合"
- ✅ "从实测数据看，[车型A]在[维度]上更有优势"
- ✅ "建议结合自身情况选择，以上仅供参考"
