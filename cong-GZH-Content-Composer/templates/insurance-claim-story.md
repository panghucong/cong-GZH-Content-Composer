# 理赔案例故事模板（T-INS-003）

> **模板类型**：故事叙事类
> **适用场景**：理赔案例分享/真实故事/情感共鸣
> **质量提升**：+20-25%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[真实案例]：[用户A]的[理赔类型]理赔经历
```

### 1.2 变体公式
```
① "理赔实录｜[疾病名称]，[理赔金额]元，[用时]天到账"
② "我的[理赔类型]理赔经历：从申请到到账，全记录"
③ "[用户A]的[理赔类型]理赔：幸好买了[产品名称]"
④ "理赔故事｜[数字]个真相，看完你也会想买保险"
```

### 1.3 关键词库
- 真实词：实录、真实案例、全记录、亲历、故事
- 情感词：幸好、后悔、庆幸、感恩、触动

---

## 二、正文结构（5段式）

### 2.1 开篇：引入故事（10-15%）
**目标**：用真实故事吸引注意力，建立共情

**模板**：
```
今天的故事，来自我的客户[用户A]（应本人要求匿名）。
[时间]，她被确诊[疾病名称]。
从确诊到理赔款到账，一共[用时]天。
理赔金额：[金额]元。

她跟我说："幸好听了你的话，买了[产品名称]。
不然这次，真的要砸锅卖铁了。"

我把她的故事写下来，
希望能帮更多人看清：保险不是"骗人的"，而是"救命的"。
```

---

### 2.2 经过：确诊经过（20-25%）
**目标**：描述确诊过程，制造焦虑感

**模板**：
```
## [用户A]的故事：从无症状到确诊

**[时间]，[用户A]在[体检机构]体检**
→ 发现[异常指标]
→ 医生建议：[进一步检查]

**[时间]，[用户A]在[医院名称]做进一步检查**
→ 检查结果：[确诊疾病]
→ 医生的话："[医生原话]"

**[用户A]的第一反应**：
"[第一反应，如：懵了/哭了/不敢相信]"

**[用户A]的第一担忧**：
"[第一担忧，如：治疗要花多少钱/会不会拖累家人]"
```

---

### 2.3 转折：保险理赔（30-35%）
**目标**：描述理赔过程，展示保险的价值

**模板**：

```
## 理赔过程：从申请到到账

**[时间]，[用户A]联系我**
→ 我说："别慌，你有[产品名称]，应该能赔。"
→ 我帮她准备了[理赔材料清单]

**[时间]，提交理赔申请**
→ 渠道：[线上/线下]
→ 所需材料：[列清单]

**[时间]，理赔款到账**
→ 用时：[数字]天
→ 理赔金额：[金额]元
→ [用户A]的话："[用户原话]"

**理赔关键点**：
1. 确诊即赔：[疾病名称]在保障范围内
2. 材料齐全：提前准备，不走弯路
3. 服务到位：[服务商]的绿通服务帮了大忙
```

---

### 2.4 反思：经验教训（20-25%）
**目标**：从故事中提炼教训，给用户建议

**模板**：
```

**反思1：[反思内容1]**

→ 我的建议：[建议内容]

**反思2：[反思内容2]**
→ [用户A]说："[原话]"

**送给所有读者的一句话**：
"[总结性金句]"
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "100%理赔"（承诺性表述）
- ❌ "[产品名称]最好"（绝对化用词）
- ❌ 泄露用户隐私（真实姓名、电话号码、住址）

### 4.2 推荐表述
- ✅ "在[用户A]的案例中，[产品名称]顺利理赔"
- ✅ "根据[用户A]的经历，建议..."
- ✅ "真实案例，仅供参考，具体以合同条款为准"

### 4.3 隐私保护
- ✅ 匿名处理：用"我的客户A"代替真实姓名
- ✅ 模糊处理：用"[某三甲医院]"代替具体医院名称
- ✅ 授权确认：发布前需获得用户书面授权

## 五、使用示例

### 5.1 输入
```
用户A：30岁女性，互联网公司员工
疾病名称：甲状腺癌
产品名称：重疾险（消费型）
理赔金额：50万元
理赔用时：7天
```

### 5.2 输出（部分）
```
# 真实案例：30岁女白领的甲状腺癌理赔经历

## 开篇：引入故事

今天的故事，来自我的客户A（应本人要求匿名）。

希望能帮更多人看清：保险不是"骗人的"，而是"救命的"。

## 经过：确诊经过

### A的故事：从无症状到确诊

**2025年11月，A在公司体检中发现甲状腺结节**
→ 体检报告提示："建议进一步检查"
→ 医生建议：去三甲医院做穿刺活检

**2025年12月，A在[某三甲医院]做进一步检查**

→ 医生的话："早期发现，手术切除即可，治愈率很高。"

"手术要花多少钱？会不会拖累家人？"

## 转折：保险理赔

### 理赔过程：从申请到到账

**2025年12月10日，A联系我**
→ 我说："别慌，你有消费型重疾险，应该能赔。"
→ 我帮她准备了理赔材料清单

→ 渠道：保险公司APP线上申请
→ 所需材料：确诊报告、身份证、银行卡、理赔申请书

→ 理赔金额：50万元

**反思2：不要等到"有症状"再买保险**

→ 我的建议：年轻健康时买最便宜，等到体检查出问题就晚了。

**反思3：不要只看重疾险，医疗险也要配**

→ 我的建议：重疾险+医疗险，一个都不能少。
```
