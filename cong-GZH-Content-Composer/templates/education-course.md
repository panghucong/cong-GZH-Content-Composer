# 模板：教育培训（Q7）

> **模板编号**：T-EDU-001
> **适用场景**：教育培训课程推荐、学习攻略
> **预设参数**：I4/C2/F2(P2/T1/U3)

---

## 模板结构

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | "为什么{X}%的{人群}都后悔没有早点{行动}" | "为什么90%的宝妈都后悔没有早点给孩子做启蒙" |
| 数字型 | "{数字}个方法，让孩子{成果}" | "3个方法，让孩子爱上阅读" |
| 情绪型 | "别让{遗憾}成为{后果}" | "别让错过最佳启蒙期成为你最大的遗憾" |

### 正文结构（F2：FABE框架）

```
【开头钩子】（痛点型/情感型）
"你是不是也遇到过..." / "作为{身份}，你是不是也有这样的焦虑..."

【F-特征】课程/产品核心特点
- 特点1：{教学理念/方法}
- 特点2：{课程体系/内容}
- 特点3：{师资/品牌}

【A-优势】为什么选这个
- 优势1：{数据支撑}（如：学员数/好评率）
- 优势2：{案例支撑}（如：学员成果）
- 优势3：{行业数据}（如：市场排名）

【B-利益】对孩子/家长有什么好处
- 利益1：{短期效果}
- 利益2：{长期价值}
- 利益3：{情感价值}（如：亲子关系改善）

【E-证据】权威背书
- 数据来源：{具体数据}
- 学员案例：{真实案例}
- 专家推荐：{专家评价}

【结尾引导】
关注我，回复"{关键词}"领取{免费资料/体验课}。
```

### 金句库

| 类型 | 金句 |
|------|------|
| 痛点型 | "教育最大的成本不是学费，而是错过的时间" |
| 数据型 | "研究表明，{X}岁前是{能力}发展的黄金期" |
| 情感型 | "每个孩子都是独一无二的，教育应该因材施教" |
| 行动型 | "今天给孩子的一份投资，就是明天最好的礼物" |

### 合规检查点

| 检查项 | 规则 |
|--------|------|
| 升学承诺 | ❌ 禁止"保证考上"/"100%提分" |
| 绝对化用语 | ❌ 禁止"最好的"/"第一名"/"唯一" |
| 价格误导 | ❌ 禁止"免费"但实际收费 |
| 数据来源 | ✅ 所有数据必须标注来源 |
| 案例真实 | ✅ 所有学员案例必须真实可查 |
