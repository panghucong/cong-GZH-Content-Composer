# 教育课程种草模板（T-EDU-002）

> **模板类型**：案例叙事+课程推荐
> **适用场景**：在线课程推广/学员案例分享/教育培训种草
> **质量提升**：+20-25%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[真实案例]：{学员姓名}从{初始状态}到{目标成果}，用了{课程名称}后的{X}个月
```

### 1.2 变体公式
```
① "{学员姓名}：从{痛点描述}到{成果描述}，我是怎么做到的？"
② "{学员身份}亲测：{课程名称}学了{X}天，{最亮眼成果}"
③ "宝妈分享｜{孩子姓名}在{课程名称}的{X}个月，变化太大了"
④ "后悔没早报！{学员姓名}的{课程名称}学习全记录"
⑤ "{年龄}岁还能学好{学科/技能}吗？{学员姓名}的答案太励志"
```

### 1.3 关键词库
- 转变词：从…到…、蜕变、逆袭、突破、开窍、进步
- 成果词：成绩提升、能力增长、自信、主动、爱上
- 情感词：后悔没早报、庆幸遇到、终于、感动

## 二、正文结构（8段式）

**模板**：
```

{身份背景}。
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "保证提分"/"100%有效"/"包教包会"（效果承诺）
- ❌ "考试必过"/"升学保障"（升学承诺）
- ❌ "全国最好"/"第一名"/"唯一"（绝对化用语）
- ❌ "XX学校名师"/"XX考试命题组"（未经授权背书）
- ❌ 使用未成年学员真实姓名和面部照片（隐私保护）

### 4.2 推荐表述
- ✅ "根据{学员姓名}的学习反馈…"
- ✅ "在{课程名称}学习期间，{学员姓名}取得了…"
- ✅ "学习效果因人而异，以上为真实案例分享"
- ✅ 学员案例需获得家长书面授权
- ✅ 价格如有变动以官方公布为准

### 4.3 隐私保护
- ✅ 学员用化名：如"{姓氏}同学"
- ✅ 成绩/证书等需脱敏处理
- ✅ 不泄露家庭住址、联系方式等隐私信息

## 五、使用示例

### 5.1 输入
```
学员姓名：小明
年龄：10岁/四年级
课程名称：某某少儿编程课
学习周期：3个月
初始状态：数学60分，逻辑思维差
成果：编程比赛市三等奖，数学提升至85分
```

### 5.2 输出（精华片段）
```
# 真实案例：10岁男孩从数学60分到编程比赛获奖，只用了3个月

## 小明是谁？

小明，10岁，小学四年级。
性格内向，不爱说话。

数学一直在及格线附近徘徊，
妈妈给他报了无数补习班，效果都不明显。

直到3个月前，妈妈给他报了{某某少儿编程课}。

3个月后：
数学从60分冲到85分，
还拿了市少儿编程比赛三等奖。

班主任在微信上问妈妈：
"小明这学期变化太大了，你们做了什么？"

---

## 妈妈的焦虑："为什么别人家的孩子可以，我家就不行？"

**痛点1：数学一直上不去**
→ 具体表现："计算题还行，一到应用题就懵，读不懂题意"
→ 妈妈试过：请家教、买习题册、报奥数班
→ 为什么没用："孩子说'我就是听不懂'，越逼越反感"

**痛点2：不爱动脑筋**
→ 具体表现："遇到稍微难一点的题就直接放弃，等着爸妈讲"
→ 妈妈试过：鼓励式教育、奖励机制
→ 为什么没用："没有内在驱动力，做什么都三分钟热度"

**痛点3：对学习没兴趣**
→ 具体表现："放学回家第一件事就是玩手机，一说学习就蔫"
→ 心理状态："觉得自己笨，学不好，干脆不学"
→ 转折点：妈妈在家长群看到同学在学编程，抱着试试看的态度报了体验课

---

## 小明的学习全过程：3个月逆袭

### 阶段一：从抵触到好奇（第1-2周）

**初始状态**：
→ "一听是'课'就不愿意来，硬拽着来的"
→ 第一节课后，小明说："妈妈，这个和数学课不一样！"

**小成果**：
→ 第2周自己写了一行代码，让小猫在屏幕上移动
→ 那天主动给妈妈讲了20分钟"我的作品"

...

## 小明说：
"我以前觉得数学好难，现在发现数学就是编程的基础。
编程让我知道数学有什么用，学起来就有意思多了。"
```
