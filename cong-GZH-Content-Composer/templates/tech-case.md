# 科技产品评测模板（T-TECH-002）

> **模板类型**：评测分析+购买指南
> **适用场景**：数码产品评测/科技产品对比/使用体验分享
> **质量提升**：+20-25%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[深度评测]：{产品名称}用了{X}天，{核心结论}——值不值得买看这篇就够了
```

### 1.2 变体公式
```
① "用了{X}个月的{产品名称}，说实话：{真实感受}"
② "{产品名称}深度评测：{核心优点}，但{核心缺点}让人纠结"
③ "预算{金额}元，{产品名称}值不值得入手？一篇说清楚"
④ "{产品A} vs {产品B}：差价{金额}元，到底差在哪？"
⑤ "{产品名称}真实体验：{X}个优点{X}个缺点，买前必看"
⑥ "别被广告忽悠了！{产品名称}的{X}个真实槽点"
```

### 1.3 关键词库
- 评测词：深度评测、真实体验、开箱、上手、横评、对比
- 决策词：值不值得买、推荐指数、购买建议、适合人群
- 真实词：说实话、坦白说、不吹不黑、有一说一

## 二、正文结构（7段式）

**目标**：快速亮出产品+核心结论，让读者决定要不要继续看

**模板**：
```

{一句话产品定位，如"XX品牌今年的旗舰之作"}

先说结论：

**{核心结论一句话，如"这是今年最值得买的XX产品——但有一个前提"}**

如果你正在纠结要不要入手，
或者和别人比较中选不定，
```

## 三、金句库

### 3.1 开篇金句
- "参数说得再好，都不如实际用一段时间。"
- "买不买，取决于你真正需要什么，而不是广告告诉你需要什么。"
- "没有完美的产品，只有对的产品。"

### 3.2 体验金句
- "有些东西，只有亲身用了才知道香不香。"
- "数据是冰冷的，体验是真实的。"
- "一款好产品，是让你用着用着就忘了它的存在。"

### 3.3 优缺点金句
- "说实话，是评测最基本的态度。"
- "每个产品都有取舍，看你能不能接受这个取舍。"
- "贵的不一定好，便宜的不一定差——但一分钱一分货大部分时候是对的。"

### 3.4 结尾金句
- "买前多看看，买后不后悔。"
- "消费要理性，选择要适合自己。"
- "最好的产品，是让你觉得'值'的产品。"

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "世界上最好的XX产品"（绝对化用语）
- ❌ 虚假/捏造测试数据
- ❌ 隐瞒明显的产品缺陷
- ❌ 收受品牌方利益但未标注"合作内容"
- ❌ 恶意贬低竞品以抬高评测产品

### 4.2 推荐表述
- ✅ "在我使用的{X}天中…"
- ✅ "根据我的使用体验…"
- ✅ "和同价位的{竞品}相比…"
- ✅ 如为合作内容，需在开头明确标注
- ✅ 测试数据需说明测试条件和环境

### 4.3 评测伦理
- ✅ 明确标注自费购买/厂商送测/合作评测
- ✅ 优缺点均如实呈现，不做选择性披露
- ✅ 价格信息标注时间节点，避免误导
- ✅ 对主观感受使用"我认为""个人感觉"等限定语

## 五、使用示例

### 5.1 输入
```
产品名称：某某品牌降噪耳机 Pro
品类：无线降噪耳机
售价：1299元
使用周期：2周
核心卖点：主动降噪、50小时续航、Hi-Res认证
核心槽点：通话降噪一般、佩戴2小时后耳罩略闷
竞品：索尼WH-1000XM5（1999元）、Bose QC45（1599元）
```

```
耳罩是蛋白皮材质，手感柔软。

**第一印象评分**：★★★★☆（8/10）

---

## 核心参数：一张表看懂

| 参数项 | 详情 |
|--------|------|
| **音频** | |
| 驱动单元 | 40mm动圈 |
| 频响范围 | 4Hz-40kHz |
| 音频认证 | Hi-Res Audio |
| **降噪** | |
| 降噪深度 | -40dB |
| 降噪芯片 | 自研V1芯片 |
| 通透模式 | 支持 |
| **续航** | |
| 开启降噪 | 50小时 |

| 快充 | 充10分钟，听5小时 |

## 用了14天，真实体验

### 场景一：通勤地铁

**做了什么**：每天1小时地铁通勤佩戴

**体验感受**：
→ 优点：降噪效果出乎意料，地铁轰鸣声几乎消失，能清晰听到音乐细节
→ 槽点：通透模式下风噪略明显

**一句话**："戴着它坐地铁，终于不用把音量开到最大了。"

---

## 不吹不黑：优点和缺点

### ✅ 优点

**1. 降噪表现越级**
→ 1299元的价位，降噪效果可媲美2000元级的索尼XM5

**2. 续航真的猛**
→ 50小时（开降噪）超过大多数竞品，一周一充绰绰有余

### ❌ 缺点

**1. 通话降噪一般**
→ 在嘈杂环境下对方会听到明显环境音
→ 经常打电话的用户可能需要再考虑

---

## 适合谁？不适合谁？

### ✅ 适合这些人

1. **通勤党/学生党**
   → 续航长、降噪强、价格友好
   → 推荐：标准版1299元即可

### ❌ 这些情况建议再想想

1. **经常打电话的商务用户**
   → 通话降噪确实是短板
   → 替代推荐：Bose QC45（1599元），通话质量更好
```
