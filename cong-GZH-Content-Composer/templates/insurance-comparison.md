# 保险对比分析模板（T-INS-002）

> **模板类型**：对比分析类
> **适用场景**：多产品对比/多方案对比/竞品分析
> **质量提升**：+15-20%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[产品A] vs [产品B]：到底哪个更值得买？
```

### 1.2 变体公式
```
① "[产品A]和[产品B]，我为什么推荐这款？"
② "测评｜[产品A]、[产品B]、[产品C]全面对比"
③ "别再瞎买了！[产品类型]对比指南"
④ "[产品A] vs [产品B]：一张表看懂区别"
```

### 1.3 关键词库
- 对比词：vs、对比、测评、区别、差异、哪个好
- 决策词：推荐、选择、避坑、指南、攻略

## 二、正文结构（4段式）

### 2.1 开篇：提出问题（10-15%）
**目标**：让用户意识到"我需要选择，但不知道选哪个"

**模板**：
```
最近有很多朋友问我：
"[产品A]和[产品B]，到底买哪个？"
"网上说法五花八门，越看越迷糊。"
"怕买错，又怕错过。"

今天，我就用一张表，
把[产品A]和[产品B]的区别讲清楚。
看完这篇，你就知道该怎么选了。
```

---

### 2.2 主体：对比分析（60-70%）
**目标**：用表格/列表清晰展示区别，帮助用户决策

#### 结构A：表格对比法
```
## [产品A] vs [产品B]：核心区别

| 对比维度 | [产品A] | [产品B] | 哪个好？ |
|---------|---------|---------|---------|
| **保障范围** | [描述] | [描述] | [结论] |
| **保费价格** | [描述] | [描述] | [结论] |
| **理赔门槛** | [描述] | [描述] | [结论] |
| **附加服务** | [描述] | [描述] | [结论] |
| **适合人群** | [描述] | [描述] | [结论] |

**我的建议**：
- 如果你[条件A]，选[产品A]
- 如果你[条件B]，选[产品B]
- 如果你[条件C]，两个都不推荐，建议考虑[产品C]
```

#### 结构B：场景化对比法
```
## 不同人群，怎么选？

**场景1：预算有限，只要基础保障**
→ 推荐：[产品A]
→ 理由：保费低，保障全，性价比高

**场景2：预算充足，想要全面保障**
→ 推荐：[产品B]
→ 理由：保障范围更广，服务更好

**场景3：有特殊需求（如[需求C]）**
→ 推荐：[产品C]
→ 理由：专门针对[需求C]设计
```

---

### 2.3 转折：避坑指南（15-20%）
**目标**：告诉用户"选产品时要注意什么"

**模板**：
```
## 买[产品类型]，这3个坑别踩

❌ **坑1：[常见误区1]**
→ 正确做法：[建议]

❌ **坑2：[常见误区2]**
→ 正确做法：[建议]

❌ **坑3：[常见误区3]**
→ 正确做法：[建议]

**记住一句话**：[总结性建议]
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "最好"、"第一"、"唯一"（绝对化用词）
- ❌ "100%理赔"、"肯定赔"（承诺性表述）
- ❌ "比[竞品]好"（贬低同业）

### 4.2 推荐表述
- ✅ "根据您的需求，更适合[产品A]"
- ✅ "从数据看，[产品A]在[维度]上更有优势"
- ✅ "建议结合自身情况选择"

## 五、使用示例

### 5.1 输入
```
产品类型：重疾险
对比产品：消费型重疾险 vs 返还型重疾险
目标用户：30岁左右上班族
```

### 5.2 输出（部分）
```
# 消费型重疾险 vs 返还型重疾险：到底哪个更值得买？

## 开篇：提出问题

最近有很多朋友问我：
"消费型和返还型，到底买哪个？"
"网上说法五花八门，越看越迷糊。"
"怕买错，又怕错过。"

今天，我就用一张表，
把消费型和返还型的区别讲清楚。
看完这篇，你就知道该怎么选了。

## 主体：对比分析

| 对比维度 | 消费型重疾险 | 返还型重疾险 | 哪个好？ |
|---------|--------------|--------------|---------|
| **保障范围** | 100种重疾+50种轻症 | 80种重疾+30种轻症 | 消费型胜出 |
| **保费价格** | 3000元/年（30岁女，50万保额） | 12000元/年（同条件） | 消费型胜出 |
| **理赔门槛** | 确诊即赔 | 确诊即赔 | 平手 |
| **满期返还** | 不返还 | 70岁返还120%保费 | 返还型胜出 |
| **适合人群** | 预算有限，要高保额 | 预算充足，要强制储蓄 | 看需求 |

**我的建议**：
- 如果你预算有限，但要高保额 → 选消费型
- 如果你预算充足，想要强制储蓄 → 选返还型
- 如果你只想买对的，不想交"智商税" → 建议消费型+自己理财

## 转折：避坑指南

### 买重疾险，这3个坑别踩

❌ **坑1：盲目追求返还**
→ 正确做法：先计算IRR（内部收益率），往往不如自己理财

❌ **坑2：只看保额，不看条款**
→ 正确做法：重点看"什么样的情况赔"、"什么样的病不赔"

❌ **坑3：一次性买够，不看通胀**
→ 正确做法：分阶段配置，每5年调整一次

**记住一句话**：保险是买保障，不是买理财。

## 结尾：行动召唤

没有"最好"的重疾险，只有"最适合"的重疾险。

如果你还在纠结，
不如点击下方『阅读原文』，
免费获取你的专属方案。

我会根据你的年龄、收入、家庭情况，
给你最中肯的建议。

**承诺**：不推销，不忽悠，只给真话。
```
