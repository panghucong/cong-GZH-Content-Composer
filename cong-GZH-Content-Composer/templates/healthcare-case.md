# 医疗种草/案例模板（T-MED-002）

> **模板类型**：案例叙事+种草推荐
> **适用场景**：医疗健康产品推荐/患者康复案例/医疗服务体验分享
> **质量提升**：+20-25%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[真实案例]：[用户姓名]的{疾病/健康问题}康复之路，{核心数据}天后发生了{变化}
```

### 1.2 变体公式
```
① "被{疾病}折磨了{X}年，{产品/服务}让我{改变}了"
② "真实分享｜{用户年龄}岁{用户身份}的{健康问题}改善全记录"
③ "医生说我{严重程度}，用了{产品/服务}{X}周后，复查结果让医生都惊讶"
④ "别再{错误做法}了！{产品/服务}用真实案例告诉你正确方法"
⑤ "{用户身份}亲测：{产品/服务}用了{X}天，{效果描述}"
```

### 1.3 关键词库
- 真实词：亲测、实录、真实案例、全记录、复诊结果、体检数据
- 情感词：终于、放心、安心、不再、告别、重获、惊喜
- 数据词：前后对比、指标变化、检查报告、第X天

## 二、正文结构（7段式）

### 2.1 封面标题段（5-8%）
**目标**：一眼抓住目标用户，建立信任感

**模板**：
```
{用户姓名}，{年龄}岁，{身份/职业}。
被{健康问题}困扰了{X}年/{X}个月。

曾经{痛苦描述}，
现在{改善结果}。

从{初始状态}到{当前状态}，
他只用了{X}周/{X}天。

今天把她的真实经历分享出来，
希望能帮到更多还在迷茫的人。
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "保证治愈"/"100%有效"/"包治百病"（疗效承诺）
- ❌ "无需看医生"/"可以替代药物治疗"（替代医疗建议）
- ❌ "经过临床试验证明"（除非有真实批文）
- ❌ 使用患者真实姓名、正面面部照片（隐私保护）
- ❌ "XX医院/XX医生推荐"（未经授权的背书）

### 4.2 推荐表述
- ✅ "在{用户姓名}的使用体验中…"
- ✅ "根据{用户姓名}的反馈…"
- ✅ "以上为个人使用体验，效果因人而异"
- ✅ "如有健康问题，请及时就医，遵从医嘱"
- ✅ "本文仅作体验分享，不构成医疗建议"

### 4.3 隐私保护
- ✅ 匿名处理：用"{姓氏}女士/先生"或化名
- ✅ 模糊处理：用"{某三甲医院}"代替具体医院名称
- ✅ 关键信息脱敏：检查报告打码、不泄露个人联系方式

## 五、使用示例

### 5.1 输入
```
用户姓名：李女士
年龄：45岁
身份：中学教师
健康问题：长期失眠（3年）
产品/服务：某品牌睡眠调理方案
使用周期：6周
核心数据：入睡时间从2小时缩短至20分钟，深度睡眠从30分钟增至2.5小时
```

### 5.2 输出（精华片段）
```
# 真实案例：45岁教师被失眠折磨3年，用这个方法6周后，终于睡了个好觉

## 李女士的故事

李老师，45岁，在一所中学教语文。
被失眠折磨了整整3年。

以前一躺下就精神，翻来覆去到凌晨两三点，
第二天顶着黑眼圈去上课，站着都能睡着。

现在晚上10点躺下，20分钟就睡着了，
一觉到天亮，醒来精神特别好。

从"失眠星人"到"秒睡体质"，
她只用了6周。

---

## 李老师的困境：为什么我比别人更容易失眠？

**痛点1：入睡困难**
→ 具体表现："躺下至少要1-2小时才能睡着，脑子里全是第二天课的内容"
→ 尝试过的办法：褪黑素、热牛奶、泡脚
→ 为什么没用："吃完褪黑素能睡，但第二天昏昏沉沉，课都没法上"

**痛点2：半夜易醒**
→ 具体表现："凌晨三四点必醒，醒了就再也睡不着"
→ 尝试过的办法：安眠药、艾灸
→ 为什么没用："安眠药不敢长期吃，怕依赖"

**痛点3：醒来疲劳**
→ 具体表现："就算睡了，也像没睡一样，浑身没劲"
→ 心理状态："觉得自己这样下去，早晚要出问题"
→ 转折点：年级组同事推荐了{睡眠调理方案}

---

## 遇见{睡眠调理方案}之后：6周全记录

**第1周：适应期**
→ 做了什么：按照方案调整作息，开始睡前放松训练
→ 感受："第一天还是不太适应，但感觉有了盼头"
→ 关键数据：入睡时间从120分钟 → 90分钟

**第2-4周：变化期**
→ 做了什么：加入饮食调理+睡前熏香
→ 感受："第三周开始，躺下半小时内能睡着了！"
→ 关键数据：入睡时间稳定在30分钟以内

**第6周后：稳定期**
→ 做了什么：维持规律作息，自如入睡
→ 最终结果："现在倒头就睡，一觉到天亮，上课再也不犯困了"
→ 关键数据：入睡时间20分钟，深度睡眠2.5小时

**李老师说**：
"3年了，终于又一次体会到'睡到自然醒'是什么感觉。"
```
