# 房地产置业指南模板（T-RE-001）

> **模板类型**：指南攻略类
> **适用场景**：买房攻略/区域分析/楼盘测评
> **质量提升**：+10-15%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[城市/区域][楼盘名称]测评：值不值得买？
```

### 1.2 变体公式
```
① "[城市]买房攻略｜[区域]板块深度分析"
② "[楼盘名称]值得买吗？3个理由告诉你"
③ "刚需买房｜[预算]元，在[城市]怎么选？"
④ "[区域]vs[区域]：哪个更有潜力？"
```

### 1.3 关键词库
- 购房词：买房、置业、楼盘、学区、交通
- 决策词：测评、攻略、分析、潜力、避坑

---

## 二、正文结构（5段式）

### 2.1 开篇：场景共鸣（10-15%）
**目标**：让读者产生"我也在纠结这个问题"的感觉

**模板**：
```
最近有很多朋友问我：
"[城市]买房，[区域A]和[区域B]，选哪个？"
"预算[金额]元，刚需首套，怎么选？"
"看了[楼盘名称]，有点心动，但怕踩坑。"

今天，我就以[典型购房者]为例，
把[城市/区域]的买房攻略讲清楚。
看完这篇，你就知道该怎么选了。
```

---

### 2.2 分析：区域/楼盘分析（30-35%）
**目标**：从多维度分析，给出专业建议

**分析维度**：
```
## [区域/楼盘名称]深度分析

**1. 地理位置**：
→ [描述位置、交通、通达性]

**2. 配套设施**：
→ [学校、医院、商场、公园等]

**3. 价格分析**：
→ [单价、总价、首付、月供]
→ [对比周边楼盘价格]

**4. 发展潜力**：
→ [规划利好、产业导入、人口流入]

**5. 风险提示**：
→ [不利因素、交付风险、质量口碑]
```

---

### 2.3 对比：多方案对比（25-30%）
**目标**：帮助用户做决策

**模板**：
```
## [区域A] vs [区域B]：哪个更适合你？

| 对比维度 | [区域A] | [区域B] | 推荐指数 |
|---------|---------|---------|---------|
| **价格** | [描述] | [描述] | [结论] |
| **交通** | [描述] | [描述] | [结论] |
| **配套** | [描述] | [描述] | [结论] |
| **潜力** | [描述] | [描述] | [结论] |
| **风险** | [描述] | [描述] | [结论] |

**我的建议**：
- 如果你是[人群A]，选[区域A]
- 如果你是[人群B]，选[区域B]
- 如果你是[人群C]，建议再等等/看看其他区域
```

---

### 2.4 避坑：购房注意事项（15-20%）
**目标**：告诉用户"买房要注意什么"

**模板**：
```
## 买[城市/区域]房，这3个坑别踩

❌ **坑1：[常见误区1，如只看价格不看地段]**
→ 正确做法：[建议]

❌ **坑2：[常见误区2，如轻信销售承诺]**
→ 正确做法：[建议]

❌ **坑3：[常见误区3，如忽略交付风险]**
→ 正确做法：[建议]

**记住一句话**：[总结性建议]
```

---

### 2.5 结尾：行动召唤（5-10%）
**目标**：引导用户咨询/看房

**模板**：
```

不如点击下方『阅读原文』，

**承诺**：不推销，只给真话。
```

- "理性购房，从容生活。"

---

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "最值得买"、"100%升值"（绝对化用词）
- ❌ "肯定涨"、"稳赚不赔"（承诺性表述）
- ❌ 贬低竞品楼盘（违反广告法）

### 4.2 推荐表述
- ✅ "根据您的需求，[楼盘A]更适合"
- ✅ "从数据看，[区域A]在[维度]上更有潜力"
- ✅ "建议结合自身情况选择，以上仅供参考"
