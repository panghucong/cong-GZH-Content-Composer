# 模板：保险种草（Q1）

> **模板编号**：T-INS-001
> **适用场景**：保险产品推荐、方案对比
> **预设参数**：I1/C2/F2(P1/T1/U3)

---

## 模板结构

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | "{数字}个{用户}不知道的{产品}真相" | "3个宝妈不知道的重疾险真相" |
| 数字型 | "{产品}配置的{数字}个关键，第{N}个最重要" | "宝宝保险配置的3个关键，第2个最重要" |
| 情绪型 | "千万别{错误行为}，否则{后果}" | "千万别给宝宝只买意外险，否则后悔都来不及" |

### 正文结构（F2：FABE框架）

```
【开头钩子】（痛点型）
"你是不是也遇到过..." / "90%的{用户}都不知道..."

【F-特征】产品核心特点
- 产品A：{特点1}
- 产品B：{特点2}
- 对比：{关键差异}

【A-优势】为什么选这个
- 优势1：{数据支撑}
- 优势2：{案例支撑}
- 优势3：{行业数据}

【B-利益】对用户有什么好处
- 利益1：{具体场景}
- 利益2：{量化结果}
- 利益3：{情感价值}

【E-证据】权威背书
- 数据来源：{具体数据}
- 案例支撑：{真实案例}
- 行业排名：{排名/评级}

【结尾引导】
关注我，回复"{关键词}"获取{专属方案/免费咨询}
```

### 金句库

| 类型 | 金句 |
|------|------|
| 痛点型 | "买保险最怕的不是贵，而是买错了还不知道" |
| 数据型 | "据统计，{X}%的家庭保险配置都存在至少1个漏洞" |
| 情感型 | "给孩子最好的礼物，不是玩具，而是一份安心" |
| 行动型 | "今天花10分钟配置好保险，明天就多一份从容" |

### 合规检查点

| 检查项 | 规则 |
|--------|------|
| 收益承诺 | ❌ 禁止"保证收益"/"稳赚不赔" |
| 绝对化用语 | ❌ 禁止"最全面"/"最好"/"唯一" |
| 理赔承诺 | ❌ 禁止"100%理赔"/"轻松理赔" |
| 对比贬低 | ❌ 禁止"XX公司很差"/"XX产品是垃圾" |
| 数据来源 | ✅ 所有数据必须标注来源 |
