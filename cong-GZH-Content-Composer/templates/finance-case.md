# 金融理财案例模板（T-FIN-002）

> **模板类型**：案例诊断+方案推荐
> **适用场景**：投资案例分享/理财规划故事/资产配置分析
> **质量提升**：+20-25%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[案例]：{用户身份}从{初始资产}到{目标资产}，用了{核心方法}，{用时}
```

### 1.2 变体公式
```
① "月入{金额}的家庭，怎么在{X}年内攒下{金额}？真实案例分享"
② "{用户身份}的理财诊断书：{核心问题}，3步调整后{变化}"
③ "年收入{金额}，存款只有{金额}？这份诊断报告太扎心但也太有用"
④ "从{亏损描述}到{盈利描述}，{用户身份}只改了这一个习惯"
⑤ "{用户年龄}岁{身份}的资产配置方案，看完直接抄作业"
```

### 1.3 关键词库
- 诊断词：诊断、复盘、梳理、漏洞、优化、调整
- 方案词：配置、方案、策略、组合、规划
- 数据词：年化、收益率、资产结构、现金流

## 二、正文结构（7段式）

### 2.1 封面段（5-8%）
**目标**：抛出案例吸引目标人群，建立"我也这样"的代入感

**模板**：
```
{用户姓名}，{年龄}岁，{职业/身份}，
{家庭情况简述}。

年收入{金额}元，
忙了{X}年，存款{金额}元。

{核心矛盾一句话描述，如："挣得不少，就是存不下来"}

直到去年，他找到我做了一次完整的财务诊断，
{X}个月后，账户余额从{金额}变成了{金额}。

今天我把他的诊断过程和调整方案，
完整分享出来。
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "保证收益"/"稳赚不赔"/"年化XX%"（收益承诺）
- ❌ 推荐具体股票代码/基金代码（投资建议）
- ❌ "XX产品一定涨"/"XX是下一个风口"（预测性表述）
- ❌ 用个人案例暗示"你也可以获得同样收益"
- ❌ 缺少"投资有风险"的标准风险提示

### 4.2 推荐表述
- ✅ "根据{用户姓名}的实际情况，我们制定了以下方案…"
- ✅ "以上数据为{用户姓名}个人案例，仅供参考"
- ✅ "过往业绩不代表未来表现"
- ✅ 使用多种资产类别进行分散化表述
- ✅ "建议根据自身风险承受能力选择"

### 4.3 数据披露
- ✅ 所有历史收益数据需标注时间范围
- ✅ 比较基准需明确说明
- ✅ 费用信息需完整披露

## 五、使用示例

### 5.1 输入
```
用户姓名：张先生
年龄：32岁
职业：互联网程序员
家庭情况：已婚，1个孩子2岁
年收入：40万
现状存款：8万（工作7年）
核心问题：高收入低储蓄
调整周期：12个月
```

### 5.2 输出（精华片段）
```
# 案例：月入3万的程序员，工作7年只存了8万，诊断后1年翻到20万

## 张先生的现状

张先生，32岁，在一家互联网公司做程序员。
已婚，孩子2岁，老婆暂时全职带娃。

一年到手约40万，
每月到手将近3万。

工作7年，存款只有8万。

他自己也纳闷："钱到底去哪了？"

---

## 张先生的财务全景：9个维度真实情况

**维度1：收入结构**
→ 主业收入：40万/年
→ 副业收入：0
→ 问题：收入来源单一，抗风险能力弱

**维度2：支出结构**
→ 房贷：8000/月
→ 生活费：12000/月（一家三口）
→ 问题：月支出2万+，收支基本打平

...

## 诊断结果：张先生的3个核心问题

**问题一：消费无预算——钱就这样"漏"掉了**
→ 每个月信用卡账单15000+
→ 问他钱花哪了，自己也想不起来
→ 如果不改：继续"月光"，抗风险能力为零

**问题二：没有投资——钱在银行卡里"睡大觉"**
→ 8万全是活期存款，年利率0.3%
→ 通胀3%，每年实际亏损2400元
→ 如果不改：辛苦挣来的钱被通胀吃掉

**问题三：没有保障——家庭"裸奔"**
→ 全家只有社保，没有商业保险
→ 一旦有人生病或意外，8万存款撑不过3个月
→ 如果不改：一场大病直接"返贫"

---

## 调整方案：3步走

### 第一步：建立预算体系（0-3个月）
→ 开始记账，设定每月预算上限
→ 发工资先存5000进专用账户
→ 3个月后月结余从0元→5000元

### 第二步：建立投资组合（3-12个月）
→ 先存够3个月生活费的紧急备用金（6万）
→ 剩余资金按"50%稳健+30%成长+20%现金"配置
→ 12个月后被动收入从0→500元/月

...
```
