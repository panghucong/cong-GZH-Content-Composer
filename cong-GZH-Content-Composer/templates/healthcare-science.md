# 模板：医疗科普（Q3）

> **模板编号**：T-MED-001
> **适用场景**：医疗健康知识科普、健康养生指南
> **预设参数**：I2/C1/F4(P2/T5/U8)

---

## 模板结构

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | "你以为的{常识}，其实是{真相}" | "你以为每天喝8杯水健康，其实这3种情况不能喝" |
| 数字型 | "{数字}个{人群}必知的{健康知识}" | "5个中老年人必知的降压真相" |
| 情绪型 | "别再{错误做法}了，{后果}太严重" | "别再空腹吃药了，这3种药空腹吃伤胃" |

### 正文结构（F4：正反辩证框架）

```
【开头钩子】（数据型/反常识型）
"最新数据显示..." / "很多人不知道..."

【正方观点】（常见认知）
- 观点1：{常见认知1}
- 观点2：{常见认知2}
- 支撑：{常见认知的数据支撑}

【反方观点】（科学真相）
- 反驳1：{科学证据1}
- 反驳2：{科学证据2}
- 支撑：{权威研究/临床数据}

【辩证分析】（综合判断）
- 适用场景：{正方观点的适用情况}
- 不适用场景：{反方观点的适用情况}
- 个体差异：{提醒个体差异}

【行动建议】（可执行）
- 建议1：{具体行动}
- 建议2：{具体行动}
- 注意事项：{注意事项}

【结尾引导】
关注我，每天学点健康知识。有健康问题？留言告诉我。
```

### 金句库

| 类型 | 金句 |
|------|------|
| 反常识型 | "不是所有的{常见做法}都是对的" |
| 数据型 | "WHO数据显示，{X}%的{疾病}可以通过{方法}预防" |
| 情感型 | "健康不是一切，但没有健康就没有一切" |
| 行动型 | "从今天开始，{具体行动}，为健康投资" |

### 合规检查点

| 检查项 | 规则 |
|--------|------|
| 诊断承诺 | ❌ 禁止"一定能治好"/"100%有效" |
| 处方建议 | ❌ 禁止直接推荐处方药 |
| 夸大功效 | ❌ 禁止"包治百病"/"药到病除" |
| 数据来源 | ✅ 所有数据必须标注来源（论文/WHO/卫健委） |
| 免责声明 | ✅ 必须包含"本文仅供参考，不构成医疗建议" |
