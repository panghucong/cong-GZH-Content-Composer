# 家庭保障规划模板（T-INS-004）

> **模板类型**：规划方案类
> **适用场景**：家庭保障方案/保障缺口分析/定制化建议
> **质量提升**：+15-20%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[年龄段]家庭保障规划：[预算]元，如何配置最合理？
```

### 1.2 变体公式
```
① "[家庭成员构成]，一份完整的保障方案"
② "保障缺口分析｜你的家庭，真的有保障吗？"
③ "[预算]元，给[年龄段]家庭的保障配置建议"
④ "家庭保障规划：从[人数]口人，到[保障类型]全覆盖"
```

### 1.3 关键词库
- 规划词：保障规划、配置方案、缺口分析、家庭保障
- 定制词：量身定制、专属方案、根据你的情况

## 二、正文结构（5段式）

### 2.1 开篇：提出问题（10-15%）
**目标**：让用户意识到"我的家庭有保障缺口"

**模板**：
```
最近有很多[年龄段]的朋友问我：
"我[年龄]岁，[家庭构成]，该买什么保险？"
"预算[金额]元，怎么配置最合理？"
"怕买错，又怕买不够。"

今天，我就以[典型家庭]为例，
给你们一份完整的家庭保障规划方案。

看完这篇，
你就知道自己的家庭，保障缺口在哪里了。
```

---

### 2.2 分析：保障缺口（20-25%）
**目标**：分析典型家庭的保障缺口

**模板**：
```
## [典型家庭]的保障缺口

**家庭情况**：
- 丈夫：[年龄]岁，[职业]，年收入[金额]
- 妻子：[年龄]岁，[职业]，年收入[金额]
- 子女：[年龄]岁，[情况]
- 老人：[年龄]岁，[情况]
- 房贷：[金额]元，[还款年限]年

**保障缺口分析**：
1. **经济支柱风险**：丈夫/妻子如果发生重疾/意外，收入中断怎么办？
2. **医疗费用风险**：生病住院，医保报销后，自费部分谁来承担？
3. **子女教育风险**：如果父母发生风险，孩子的教育金谁来保证？
4. **老人赡养风险**：如果父母生病，医疗费用谁来承担？

**结论**：
这个家庭的保障缺口，至少有[金额]元。
```

### 2.3 方案：配置建议（35-40%）
**目标**：给出具体的配置方案

**模板**：
```

### 丈夫（经济支柱）
- **重疾险**：[保额]万，保障[年限]，保费[金额]元/年

- **定期寿险**：[保额]万，保障到[年龄]岁，保费[金额]元/年
- **意外险**：[保额]万，保费[金额]元/年
- **小计**：[金额]元/年

### 妻子（经济支柱）
- **重疾险**：[保额]万，保障[年限]，保费[金额]元/年
- **医疗险**：百万医疗，保费[金额]元/年
- **定期寿险**：[保额]万，保障到[年龄]岁，保费[金额]元/年
- **意外险**：[保额]万，保费[金额]元/年
- **小计**：[金额]元/年

### 子女（可选）
- **重疾险**：[保额]万，保障[年限]，保费[金额]元/年
- **医疗险**：少儿医疗，保费[金额]元/年
- **小计**：[金额]元/年

### 老人（可选）
- **防癌险**：[保额]万，保费[金额]元/年
- **意外险**：[保额]万，保费[金额]元/年
- **小计**：[金额]元/年

**总计**：[金额]元/年，占家庭年收入[百分比]%
```

### 2.4 优化：调整建议（15-20%）
**目标**：根据预算调整方案

**模板**：
```
## 预算不够？这样调整

**情况1：预算充足（[金额]元以上）**
→ 建议：按上述方案配置，保障全面

**情况2：预算有限（[金额]元-[金额]元）**
→ 建议：优先配置经济支柱的重疾险+医疗险+定期寿险
→ 子女和老人的保障，可以后续逐步补充

**情况3：预算紧张（[金额]元以下）**
→ 建议：优先配置经济支柱的医疗险+意外险（保费低，保障高）
→ 重疾险和定期寿险，等收入提高后再补充

**记住**：保险配置不是"一步到位"，而是"逐步完善"。
先有保障，再求全面。
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "最好"、"第一"、"唯一"（绝对化用词）
- ❌ "100%理赔"、"肯定赔"（承诺性表述）
- ❌ "必须买[产品名称]"（强制推荐）

### 4.2 推荐表述
- ✅ "根据[_s_家庭情况]，建议配置..."
- ✅ "如果预算有限，可以优先配置..."
- ✅ "以上方案仅供参考，具体以合同条款为准"

## 五、使用示例

### 5.1 输入
```
家庭类型：30岁夫妻+1个孩子+双方父母
丈夫：30岁，互联网公司，年收入40万
妻子：28岁，教师，年收入20万
子女：2岁
老人：双方父母65岁左右，有养老金
房贷：200万，30年
预算：3万元/年
```

### 5.2 输出（部分）
```
# 30岁夫妻家庭保障规划：3万元，如何配置最合理？

## 开篇：提出问题

最近有很多80后、90后的朋友问我：
"我30岁，夫妻双职工，有娃有房贷，该买什么保险？"
"预算3万元，怎么配置最合理？"
"怕买错，又怕买不够。"

今天，我就以[30岁夫妻+1个孩子+双方父母]为例，
给你们一份完整的家庭保障规划方案。

看完这篇，
你就知道自己的家庭，保障缺口在哪里了。

## 分析：保障缺口

### 典型家庭的保障缺口

**家庭情况**：
- 丈夫：30岁，互联网公司，年收入40万
- 妻子：28岁，教师，年收入20万
- 子女：2岁
- 老人：双方父母65岁左右，有养老金
- 房贷：200万，30年

**保障缺口分析**：
1. **经济支柱风险**：丈夫/妻子如果发生重疾/意外，收入中断怎么办？
   → 缺口：至少100万（覆盖房贷+子女教育+老人赡养）
2. **医疗费用风险**：生病住院，医保报销后，自费部分谁来承担？
   → 缺口：至少200万（重疾治疗+康复费用）
3. **子女教育风险**：如果父母发生风险，孩子的教育金谁来保证？
   → 缺口：至少100万（到18岁的教育费用）
4. **老人赡养风险**：如果父母生病，医疗费用谁来承担？
   → 缺口：视父母医保情况而定

**结论**：
这个家庭的保障缺口，至少有400万元。
```
