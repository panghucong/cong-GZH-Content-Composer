# 旅游攻略种草模板（T-TRAVEL-001）

> **模板类型**：种草攻略类
> **适用场景**：旅游攻略/目的地推荐/旅行避坑
> **质量提升**：+10-15%
> **结构版本**：v1.0.0

---

## 一、标题公式

### 1.1 标准公式
```
[目的地]旅游攻略：[天数]天[花费]元，超详细！
```

### 1.2 变体公式
```
① "[目的地]值得去吗？玩了[天数]天，告诉你真相"
② "旅游种草｜[目的地]，此生必去的[理由]"
③ "[目的地]vs[目的地]：哪个更适合[人群]？"
④ "旅行避坑｜去[目的地]，这3个坑别踩"
```

### 1.3 关键词库
- 旅游词：旅游、旅行、攻略、种草、必去
- 实用词：避坑、省钱、打卡、美食、住宿

## 二、正文结构（5段式）

### 2.1 开篇：场景共鸣（10-15%）
**目标**：让读者产生"我也想去"的感觉

**模板**：
```
你有没有过这种感觉：
翻开朋友圈，看到别人在[目的地]的美照，
心里默默想："我也想去。"

但转念一想：
"听说[目的地]很坑？"
"[目的地]真的值得去吗？"
"[预算]元，够不够玩？"

今天，我就以[典型游客]的身份，
把[目的地]的攻略讲清楚。
看完这篇，你就知道该不该去了。
```

---

### 2.2 攻略：详细行程（40-50%）
**目标**：给出具体可操作的行程安排

**模板**：
```
## [目的地][天数]天行程安排

**Day 1：[行程主题，如初见XX]**
→ 上午：[景点A]（门票[价格]元，游玩[时间]小时）
→ 中午：推荐餐厅[名称]（人均[价格]元）
→ 下午：[景点B]（[描述]）
→ 晚上：住宿推荐[酒店名称]（[价格]元/晚）

**Day 2：[行程主题，如深度XX]**
→ 上午：[景点C]（[描述]）
→ 中午：推荐餐厅[名称]（[描述]）
→ 下午：[活动D，如徒步/潜水/滑雪]（[描述]）
→ 晚上：[夜景/夜市]（[描述]）

**Day 3：[行程主题，如告别XX]**
→ 上午：[补打卡景点]（[描述]）
→ 中午：最后一餐[特色美食]（[描述]）
→ 下午：返程（交通[方式]，[时间]小时）

**费用清单**：
- 机票：[金额]元
- 住宿：[金额]元
- 门票：[金额]元
- 餐饮：[金额]元
- 交通：[金额]元
- **总计**：[金额]元/人
```

---

### 2.3 种草：必去理由（20-25%）
**目标**：激发读者的旅行欲望

**模板**：
```
## 为什么[目的地]值得去？

**理由1：[理由标题，如绝美自然风光]**
→ [描述，50-100字]

**理由2：[理由标题，如独特文化体验]**
→ [描述，50-100字]

**理由3：[理由标题，如性价比超高]**
→ [描述，50-100字]

**我的感受**：
"[个人感受，20-50字]"
```

---

### 2.4 避坑：旅行注意事项（15-20%）
**目标**：告诉读者"旅行要注意什么"

**模板**：
```
## 去[目的地]，这3个坑别踩

❌ **坑1：[常见误区1，如不去景点被骗]**
→ 正确做法：[建议]

❌ **坑2：[常见误区2，如餐饮被宰]**
→ 正确做法：[建议]

❌ **坑3：[常见误区3，如住宿条件差]**
→ 正确做法：[建议]

**记住一句话**：[总结性建议]
```

---

### 2.5 结尾：行动召唤（5-10%）
**目标**：引导读者行动（收藏/转发/咨询）

**模板**：
```
## 最后说一句

[目的地]，值得你去一次。

如果你也心动了，
不妨点赞+收藏这篇攻略，
下次旅行，就选它了。

**P.S.** 你在[目的地]有什么特别经历？
欢迎在评论区分享你的故事。

---
P.P.S. 前100名转发本篇文章的朋友，
免费赠送《[目的地]深度旅行手册》（价值99元）。
```

## 四、合规检查点

### 4.1 必须避免的表述
- ❌ "最美"、"第一"、"唯一"（绝对化用词）
- ❌ "100%安全"、"肯定好玩"（承诺性表述）
- ❌ 虚假宣传景点/活动（误导消费者）

### 4.2 推荐表述
- ✅ "[目的地]给我留下了深刻印象"
- ✅ "根据我的体验，[景点A]值得一去"
- ✅ "以上仅供参考，具体以实际情况为准"
