# cong-GZH-Content-Composer

公众号推文 AI 全流程生成器 —— 从一句话到排版就绪，一个 Skill 搞定。


---

## ✨ 它不是什么“AI 帮你写文章”的工具
它是 **一套完整的公众号内容生产线**，从生成到合规、再到排版，一键搞定。

- 📚 **6 种生成模式**：快速 / 标准 / 改写 / 批量 / 快捷 / 仅排版
- 🏥 **8 个行业深度适配**：保险 / 医疗 / 金融 / 教育 / 科技 / 房地产 / 汽车 / 旅游
- 🛡️ **4 层合规风控**：通用广告法 + 行业敏感词 + 语境豁免 + 自定义词库
- 🎯 **质量门控系统**：初稿 < 80 分自动重写，终稿 ≥ 92 分才输出
- 💖 **情感共鸣引擎**：五阶段情感曲线 + Pain-Relief 叙事 + CTA 三件套
- 🎨 **18 种 HTML 组件**：section-only 架构，3 色系统，直接粘贴到公众号编辑器
- ⚡ **Token 优化 70%**：分章节按需加载，共享定义库，数字编码系统

---

## 🎬 30 秒看效果
**你输入：**
> 帮我写一篇重疾险异议处理话术，给保险新人看的

**它输出：**
→ 自动识别「保险」行业，加载合规词库和评分权重
→ 确认目标用户和字数（一句话搞定）
→ 生成初稿 + 自评 93 分 → 合规扫描 0 命中
→ 输出浅蓝主题公众号 HTML + Markdown 备份
→ 5 个异议 × 3 步应答，全配场景 + 话术 + 原理解析

## 🔧 快速上手
1.  **准备你的 AI 环境**：支持主流大模型 API（如 OpenAI、Claude、文心一言等）
2.  **导入 Skill 文件**：将本项目的配置文件导入你的 AI 客户端
3.  **输入你的需求**：一句话说清主题、行业和目标用户
4.  **一键生成发布**：复制输出的 HTML 到公众号编辑器，直接发布
